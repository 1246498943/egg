# 概述

这是一个c++初级功能扩展库。只依赖于语言环境和系统API。不同于boost和poco这类大而全的库，egg仅仅是提供一些常用功能的轻量级实现，也提供了更简单的编译流程。

目的仅仅是为了开发方便，固定一个完善的版本。总是反复的写基础功能是非常浪费精力的事情。

合并来源 NeZha eggEngine avUnity
